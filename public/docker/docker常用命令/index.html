docker常用命令

Jun 27, 2017
1 min read

查看当前机器上的镜像：

docker images

下载镜像：

docker pull <image>

删除镜像：

docker rmi <image>

运行镜像：

docker run -d -p 3306:3306 hub.c.163.com/library/tomcat:latest

查看容器列表及状态：

docker ps -a
