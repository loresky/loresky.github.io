win

win的linux_location

谷歌搜索 win10 linux location,然后里面有 stackoverflow 的几个链接。
里面说到 linux 存放位置在
C:\Users{user}\AppData\Local\lxss{username}
其中 {user} 指的是 windows 的用户，{username} 指的是 linux 里的用户。
以及你的 /home 文件夹会在
C:\Users{user}\AppData\Local\lxss\home\
